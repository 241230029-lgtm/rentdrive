# RentDrive

RentDrive adalah aplikasi manajemen rental kendaraan berbasis web yang dibangun menggunakan Laravel, Inertia.js, React, Tailwind CSS, dan MySQL.

## Fitur Utama

### Pelanggan

- Melihat katalog kendaraan
- Melakukan booking kendaraan
- Mengunggah dan memperbarui identitas
- Melakukan pembayaran
- Melihat status penyewaan
- Melihat riwayat transaksi
- Melihat detail pengembalian dan denda
- Mengelola profil akun

### Admin

- Mengelola data kendaraan
- Mengelola dan menyetujui booking
- Memverifikasi identitas pelanggan
- Mengelola pembayaran
- Memproses pengembalian kendaraan
- Menginput kondisi dan kerusakan kendaraan
- Menghitung denda keterlambatan dan kerusakan
- Melihat riwayat transaksi
- Melihat laporan operasional

### Owner

- Melihat dashboard bisnis
- Melihat pendapatan
- Melihat jumlah transaksi
- Melihat kendaraan terlaris
- Melihat laporan bisnis berdasarkan periode

## Teknologi

- Laravel
- PHP
- MySQL
- Inertia.js
- React
- Tailwind CSS
- Vite
